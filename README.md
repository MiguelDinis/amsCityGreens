# amsCityGreens

## Install

Clone this repository
Create a virtual environment:

```bash
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```

## How to run:
Open one terminal, and execute:

```bash
$ source venv/bin/activate
$ python3 manage.py runserver
```
Open browser with ip 127.0.0.1:8000
